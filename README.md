# エレベーターピッチ

「GEN」というサービスは、
限られた時間の中で、満足する漫画作品を描きたい
「同人作家さん」に向けた「漫画の進捗管理アプリ」です。
他のアプリと違って、「ページごと」に状況を記録します。
また、「力を入れたいページ」にフラグをつけられます。
さらに、いきづまったら「保留」して、作業を切り替えることができます。

## このアプリでできないこと
- 作業時間を記録できません。
  - きがるに作業を開始・中断できます。
